bggsec é um Unified Tool Manager focado em Kali / Pentest, criado para organizar, instalar, atualizar e ajudar a executar ferramentas de segurança de forma rápida e simples — tudo a partir de um único menu em Bash.







Projeto educacional, ideal para estudantes de CyberSecurity e para quem quer manter o ambiente de pentest organizado.
